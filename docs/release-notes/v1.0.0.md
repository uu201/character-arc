# CharacterArc v1.0.0

> 首个正式版本 · Released 2026-05-24

弧光（CharacterArc）是一款本地优先的 AI 小说创作桌面应用，围绕项目设定、章节写作与 AI 协作打造一体化工作台。

---

## ✨ Features

### 项目与资料

- **项目中心**：创建、查看、编辑、删除小说项目，互相隔离
- **新建项目向导**：填写题材/篇幅/简介后，可由 AI 螺旋式生成首批设定与大纲（骨架 → 展开 → 校验）
- **知识中心**：沉淀项目事实、创作记忆、参考资料与风格分析结果
- **拆书知识库**：全局共享，与项目解耦，支持参考作品深度拆解

### 世界观与结构

- 世界观、角色、组织、关系四类设定资产管理
- **关系图谱**：基于 Cytoscape 的可视化，呈现角色与组织关联
- **剧情大纲**：双栏交错时间线布局，按分卷组织节点，支持拖拽排序与 AI 扩写
- **剧情线索**：维护伏笔、悬念与回收计划

### 章节创作

- **VS Code 风格三栏工作区**：目录树 + TipTap 正文编辑器 + AI 侧边栏
- 自动保存与历史快照，支持手动版本回滚
- 阅读模式 / 专注模式 / 字数目标跟踪
- 导出 `.txt` / `.docx`，工作区可导出 JSON 快照

### AI 辅助

- 章节润色、续写、改写、节奏调整、摘要、伏笔识别、后续剧情链
- 章节初稿整章单次流式生成，引入写前备忘与写后审计
- AI 对话流式输出，生成中支持中途停止
- **Agent Loop 模式**：模型按 Skill 索引与工具注册表循环思考，可直接调用工具编辑章节
- **任务进度面板**：统一查看运行中与历史 AI 任务

### 封面工作台

- 面向番茄、起点、晋江、知乎盐言、七猫、刺猬猫等平台生成封面 Prompt
- 调用图像模型生成预览，支持历史版本对比

### 多厂商接入

- DeepSeek（含 `reasoning_content` 推理回传）、通义千问、智谱 GLM、Kimi、SiliconFlow、OpenAI、Anthropic、Ollama
- 支持 New API / One API 自定义网关
- Embedding 模型可独立配置

### 内置 Skill 系统

预置 15 个 Skill，覆盖长篇/短篇/章节/风格/封面等场景：

| 分类 | Skill |
|---|---|
| 长篇 | `story-long-write`、`story-long-analyze`、`story-long-scan` |
| 短篇 | `story-short-write`、`story-short-analyze`、`story-short-scan` |
| 章节 | `story-chapter-exec`、`story-chapter-repair` |
| 风格 | `humanizer-zh`、`style-fingerprint`、`style-fusion` |
| 其他 | `story-cover`、`story-deslop`、`story-blueprint`、`story-format-tomato` |

- Skill 元数据基于 frontmatter，支持项目级独立导入
- 章节写作时自动按任务匹配最合适的 Skill 包

## 🏗️ Architecture

- 基于 **Electron + Vue 3 + TypeScript**，构建工具 electron-vite (Vite 7)
- 主进程统一管理 SQLite 持久化与 AI 调用
- AI 管线迁移至 **Vercel AI SDK**，支持流式、Agent、工具调用
- 向量索引与语义检索集成（导入参考小说时自动建立索引）

---

## 🎨 UI/UX

- **Nord 风格深色主题** + 暖色调浅色主题
- 自定义标题栏（titleBarOverlay），主题切换实时生效
- 章节创作界面基于 UI/UX Pro Max 规范优化
