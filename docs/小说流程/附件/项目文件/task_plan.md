# 《港综：我从烂仔到总统的一生》当前阶段任务计划

## 当前阶段目标：开书起盘期

把“炮灰顶锅死局”推进成：

- 凌夜临时脱锅
- 拿到第一笔小钱
- 扣下一张关键原始交数页
- 让 `人生系统` 第一次克制激活
- 抛出大头上面那条线

当前仍处于：`第1章待写 / 尚无正式正文`。

## 第1章关键纠偏

1. 第1章不能吃太多。
   只能完成开篇立人、破局、拿第一块小资源和留钩子，不能直接写成大上位。
2. 大头不能写蠢。
   不能让真钱、真账、真把柄全摆在一起等人拿。
3. 凌夜不能“刚穿越就看穿一切”。
   所有判断都必须有硬抓手。
4. 控场不能只靠暴力宣言。
   必须让现场的责任链、账线、上面那条线和马仔反应一起构成压力。
5. 系统第一次出现必须克制。
   只给短提示和实用奖励，不给长说明书。
6. 章末钩子不能跳太高。
   只能钩到大头上面的线，不能直接跳到更高层大佬争霸。

## 第1章硬抓手库

- 账本是誊抄页，不是原始交数页
- 账页上没有封条号、交数签押或交接痕迹
- 桌上的钱是散的，不像正式交数封包
- 大头在反复看表或看门
- 堵门站位说明今天不是“讲道理”，而是“等结果”
- 马仔的反应不像来对账，更像来做见证

## 第1章目标字数

- `3000-3400` 字

## 第1章四场修正版

### 场景1：识局与破局前奏

- 目标字数：`600-800`
- 重点：
  - 不写大时代风貌，先写逼仄空间和压迫物件
  - 大头在施压，逼凌夜认下这笔烂账
  - 凌夜刚接完记忆，脑子晕，但不慌
  - 凌夜第一句话必须是能精准打断节奏的现实反问
- 必落信息：
  - 烟灰缸
  - 堵门马仔
  - 桌上散钱或誊抄账页
  - 账页逻辑漏洞

### 场景2：掀桌子与极限施压

- 目标字数：`1000-1200`
- 重点：
  - 凌夜借信息差和责任链反掐大头
  - 一旦动手就要控场，不写花哨武打
  - 核心是：把“被审判者”变成“让别人不敢乱动的人”
- 必落信息：
  - 动作必须短、狠、直接
  - 先压住大头，再压住房里的人
  - 当场谈判只换局部成果，不换整盘

### 场景3：人生系统初显与资源落袋

- 目标字数：`600-800`
- 重点：
  - 破局刚完成，系统立刻短促亮相
  - 奖励必须实用、克制
  - 凌夜没空发呆，拿钱、拿关键页、留警告、准备走
- 必落信息：
  - 第一笔资金到账，但规模不能夸张
  - 拿到的关键材料只能是一张“原始交数页”或同级别小抓手

### 场景4：章末钩子与余波

- 目标字数：`300-400`
- 重点：
  - 凌夜推门离开，完成从待宰羔羊到吃人野狼的切换
  - 大头捂伤起身，准备把事情往上面报
  - 或者系统亮出下一步短提示
- 必落信息：
  - 钩子只到大头上面那条线
  - 不跳到更高层全盘格局

## 第1章逐段正文提纲

1. 从逼仄的后屋起手，桌面账页、散钱、烟灰缸、堵门站位一起压上来。
2. 凌夜刚接完记忆，头有点晕，但第一反应是先看人、看门、看桌面，不先慌。
3. 大头继续逼账，口径是“认了这笔账，你还能活”。
4. 凌夜用一句极现实的反问切断节奏，直接点出账页上的致命漏洞。
5. 房里短暂停顿，马仔反应和大头脸色一起证明这不是正常对账。
6. 凌夜顺着漏洞再追一层，把“大头拿他填命”的真相逼到半明面。
7. 大头想硬压，凌夜直接掀桌制造空隙，抢先手。
8. 凌夜用最直接的街头动作控住大头，不写花招，只写结果和压迫。
9. 马仔本来想动，但凌夜把责任链和“谁担这口锅”抛回去，房间里没人敢真上。
10. 凌夜逼大头当场做出局部让步，先让自己从“必死背锅人”变成“暂时不能碰的人”。
11. 凌夜只拿小钱和一张关键原始交数页，不贪大，不把整盘搬空。
12. 破局瞬间，`人生系统` 首次激活，提示极短，只给实用型奖励或认知加成。
13. 凌夜留下冷警告，推门离开，大头和马仔没人敢追。
14. 尾巴抛出大头上面那条线，形成第2章入口。

## 第2-3章任务链

### 第2章建议

- 把第1章拿到的小钱和关键页变成“能站稳第一步”的现实筹码。
- 写大头余波，不要立刻跳新地图。
- 开始试探大头上面那条线到底是谁在控这笔账。
- 若要接系统提示，第2章也只能小开口，不要直接变成任务面板流。

### 第3章建议

- 把第1章的破局，推进成“第一块真正可持续的生存空间”。
- 可以开始接触第一条更高层接口，但只能是试探或擦边，不是正式大联手。
- 第一批“人”“钱”“白线”仍要慢慢落，不要一口气全到。

## 当前风险

- 如果第1章写成单纯打架，凌夜就立不出脑子。
- 如果第1章写成单纯破案，港综压迫感会不够。
- 如果系统第一次戏份太多，会抢掉开书应有的人物戏。
- 如果章末钩子跳太高，后面两章会失去梯子感。
