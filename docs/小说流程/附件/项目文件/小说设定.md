# 小说设定总表

## 一、项目硬设定

- 书名：`港综：我从烂仔到总统的一生`
- 世界线起点：`1990`
- 世界底层：`港综 + 现实历史` 融合世界
- 主角：`凌夜`
- 主角起点：`1990年港岛底层烂仔 / 混混 + 穿越者`
- 系统：`人生系统`
- 当前真实阶段：`开书筹备期 / 第1章待写 / 尚无正式正文`

## 二、主角与系统

### 1. 凌夜

- 护短、霸道、底线灵活
- 不圣母
- 不无脑莽
- 不降智
- 动手一定考虑收尾和后患
- 强势来源于判断、抓手、筹码和压迫，不靠旁白硬吹

### 2. 人生系统

- 定位是辅助器，不代替凌夜做决定
- 第一次激活必须克制、实用、短提示
- 未正式落正文前，不写成完整说明书
- 不得提前把系统后续功能记成现成筹码

## 三、成长路线

1. 先社团立身
2. 再商业扩张
3. 再逐步往更高层级推进

## 四、第1章入口设定

- 入口固定为：`大头逼凌夜认下烂账`
- 场景重心固定为：`逼仄空间、站位压迫、账页漏洞、即时反控场`
- 第1章只允许拿到：
  - 临时脱锅
  - 一小笔钱
  - 一张关键原始交数页
  - 系统首次短促激活
  - 指向大头上面那条线的钩子

## 五、固定文风

- 轻松幽默
- 带一点聪明劲和大智慧
- 有番茄爽点，但不失现实逻辑
- 语言直接
- 多写动作，少写静态描述
- 保持叙事感，不写成散文
- 口语化、顺口、自然

## 六、核心写作心态

你不是来炫技的，也不是来交任务的。
你只是认真讲故事，把人物眼下这一步写扎实。
不要急着完成任何东西。
先把细节、动作、停顿、反应、压力和后果写够，让故事自己往前走。

## 七、去模板味与人性化规则

- 避免汇报腔、说明书腔和假完整逻辑
- 句式长短自然变化
- 少用“首先、其次、最后、总之、综上所述”这类硬套词
- 少用“值得注意的是、需要指出的是、让我们来”这类机械引导语
- 多用动作、细节、物件、停顿和口语化转折
- 情绪要真实，不要硬塞情绪词
- 第二人称统一用“你”，不用“您”

## 八、如任务是改写 / 润色 / 人性化优化

默认优先目标：

- 保住原意
- 保住核心信息
- 去掉机械味
- 提高自然度和可读性

必要时可按这些方向工作：

- 叙述性
- 描述性
- 对话性
- 正式
- 非正式
- 幽默
- 讽刺 / 吐槽
- 抒情
- 客观
- 主观
- 简洁
- 详尽
- 悬疑
- 紧张
- 平静

## 九、绝对禁止

- 文学腔和装深沉
- 散文化美文写法
- 为了推进剧情让角色集体降智
- 把黑帮、金融、军火、暗杀等元素写成现实教程
- 抄 `57小说/` 的句子、桥段顺序、对白和人物结构

## 十、说明

- `New 0111.txt` 是母设和方法汇总。
- `57小说/` 只借写法机制，不借事实。
- 当前项目一切正式事实，以最新正文和状态卡为准。
