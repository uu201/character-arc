# 常见失误回退表

用于《港综：我从烂仔到总统的一生》当前开书期。

## 第1章最常见失误

1. 开头先讲时代背景，迟迟不进冲突
2. 凌夜刚进场就像全知全能
3. 大头像傻子一样把真钱、真账、真把柄全摆出来
4. 只靠一记狠手控全场，责任链没有立起来
5. 系统第一次出现过重，直接压掉人物戏
6. 第1章一口气拿到钱、人、白线、大上位
7. 章末钩子跳太高，直接越过大头这层
8. 语言太文艺、太散、太像说明书
9. 套话太多，像AI汇报稿
10. 结尾停在半截问话，没有真正形成新的力量关系

## 回退修法

- 开头切回逼账现场
- 判断补回硬抓手
- 大头只漏一张关键原始交数页，不交整盘
- 控场时补回“谁先动谁担锅”的责任链
- 系统只保留短提示和实用奖励
- 第1章只拿小钱、小抓手、小反转
- 钩子只收在大头上面的线
- 删掉文学腔和汇报腔，改回动作、物件、停顿、反应
