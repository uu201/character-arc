# 常见失误回退表

用于《港综：我从烂仔到总统的一生》当前开书期。

## 开篇常见错误

1. 先写背景，后写冲突，导致开头发慢
2. 凌夜没有抓手就直接看穿全局
3. 大头像故意给主角送把柄
4. 动手很狠，但现场责任链没有立起来
5. 系统第一段戏写太长
6. 第1章资源到账过多
7. 章末钩子跳过大头这层，直接拔太高
8. 语言太文艺、太汇报、太像任务卡翻译

## 默认修法

- 开头直接切回逼账后屋
- 至少补 `3` 个硬抓手再让凌夜下判断
- 大头桌上只留局部漏洞，不留整盘
- 动手后立刻补回“谁先动谁担锅”的责任链
- 系统只保留短提示和实用奖励
- 第1章只拿小钱和关键页
- 钩子只收到大头上面的线
- 删掉汇报腔，改成动作、停顿、物件、反应
